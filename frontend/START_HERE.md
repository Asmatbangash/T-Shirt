# 👋 Welcome to Threadify!

## 🎉 Your Premium E-Commerce UI is Ready!

This is a complete, production-ready e-commerce interface for a custom T-shirt brand.

---

## ⚡ Quick Start (3 Steps)

### 1. Install Dependencies

```bash
cd frontend
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

### 3. Open Browser

Navigate to `http://localhost:5173`

**You'll see a page switcher at the bottom** - click any button to explore all 8 pages!
